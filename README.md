# desktop-Tamagotchi
FUN!!! tamagotchi desktop widget with application shortcuts for Rainmeter  

## Installation instructions
Copy the folder `DesktopTamagotchi` to your Rainmeter Skins location, f.e. `C:\Users\..\Documents\Rainmeter\Skins\`   
Click `Refresh all` in Rainmeter and load BIG or smalltamagotchi.ini  
  
To change icons, open .ini file (in notepad) and use existing icons of choice in assets folder, or add new image files in asset folder and refer to its path in the ini file: `ImageName=assets/... .png` (or .jpeg).  
  
Change the 'LeftMouseUpAction' to the path of the application of choice! f.e. `LeftMouseUpAction=["C:\Program Files\Mozilla Firefox\firefox.exe"]` - enjoy! with love - RosseR.
